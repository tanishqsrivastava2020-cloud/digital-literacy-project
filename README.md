## 📖 Project Overview

This project was completed as part of the CSE0001 – Digital Literacy course at VIT Bhopal University. The objective of this project is to develop essential digital skills required for students in today’s technology-driven world.

The project is structured into five tasks, each focusing on a different aspect of digital literacy such as online awareness, professional presence, communication, and cyber safety.

---
